This assignment is for shell scripting a secure vault system, store secrets, control file access, monitor security risks and also allows security updates.

Step 1 executed by the file "vault_stup.sh": is to create adirectory and files where secrets will be stored.
Step 2 executed by the file "vault_permissions.sh": is to control file access, mange each group's permissions and b able to update the permissions and also display the final permissions.
Step 3 executed by the file "vault_ops.sh": this is to be able to allow security updates, add a new secret to secret.txt or update a secret, add a timestamped log into log.txt and also Print "access denied" to whoever needs to access key.txt because it should never be shown.
Step 4 executed by file "vault_monoitor.sh": this is a step that will be used to monitor our vault where it will dispaly the file name, size in bytes, last modified date, and current permissions for each file in the secure_vault directory, and save all results to vault_report.txt.
